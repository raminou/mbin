# mbin
Nice binary display of numbers

Example:
```
python3 -m mbin 0xdeadbeef
│ 1101 │ 1110 │ 1010 │ 1101 │ 1011 │ 1110 │ 1110 │ 1111 │
└──┬───┼──┬───┼──┬───┼──┬───┼──┬───┼──┬───┼──┬───┼──┬───┤
   │  28  │  24  │  20  │  16  │  12  │   8  │   4  │   0
   d      e      a      d      b      e      e      f

Hexa:                       0x dead beef
Int:                       3 735 928 559
Float32:          -6.259853398707798e+18
```
